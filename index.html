FXForm2 by amischler

Stop coding forms: FXForm can do it for you!

FXForm2 is a library providing automatic form generation for JavaFX beans. Main features include: * CSS support * Bean Validation handling (JSR 303) * Fields reordering and filtering * Tooltips * Localization
